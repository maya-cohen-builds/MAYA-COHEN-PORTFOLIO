# Maya Cohen Portfolio

Personal portfolio site showcasing AI product marketing and GTM case studies.

Live: [mayacohenportfolio.lovable.app](https://mayacohenportfolio.lovable.app)

## What it is

A full case study portfolio built to support a technical PMM job search targeting
AI, gaming, and developer tooling companies. Features 11 project cards spanning
agentic AI systems, cloud pipeline product ownership, enterprise GTM, and
vibe-coded applications.

## Stack

- React + TypeScript
- Vite
- Tailwind CSS
- Deployed via Lovable

## Built with

Claude API · Supabase · Lovable (L2 certified)
